# Contributing

Thanks for your interest in contributing to node-boilerplate!

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/node-boilerplate.git`
3. Install dependencies: `yarn install`
4. Create a branch: `git checkout -b feat/your-feature`

## Development

```bash
yarn test   # run tests
yarn lint   # run ESLint
```

## Submitting Changes

1. Ensure tests pass and lint is clean
2. Commit with a clear message describing the change
3. Push to your fork and open a Pull Request against `main`
4. Fill in the pull request template

## Code Style

This project uses ESLint. Run `yarn lint` before committing; the pre-commit hook will also enforce it.

## Reporting Issues

Use the GitHub issue templates for bug reports and feature requests.
